Подключение рекомендуемых товаров к собственным страницам

В новой версии нашего плагина мы добавили возможность отображения рекомендуемых от Apishops товаров. И если вы уже используете нашу форму и добавите к параметрам инициализации ещё и параметр featured равным { container: '#featureconatiner', count:3 }, то в контейнере #featureconatiner наш плагин разместит три рекомендуемых товара.

Для отображения блока рекомендуемых товаров, к стандартной инициализации плагина нужно добавить всего один обязательный параметр featured.container, и он укажет нашему плагину где должны расположиться рекомендуемы товары.

Пример: 

<script>
        $('#containerid').apishopsForm({
            type:'inline', /*тип открытия [inline|modal]*/
            form:'light', /*тип формы [normal|light|jquery-selector]*/
            siteId:147233, /*айди сайта из панели партнера http://www.apishops.com/Webmaster/WebsiteGroup/WebsiteGroupList.jsp */
            productId:888647,/*айди товара из ассортимента на странице http://www.apishops.com*/
            featured:{
               container: '#featureconatiner', /*контейнер для размещения в нем рекомендуемых товаров*/
            }
        });
</script>
<div id=containerid>
    Контейнер для размещения основного товара
</div>

<div id=featureconatiner>
    Контейнер для размещения рекомендуемых товаров
</div>

Внимание: по умолчанию(без паарметров featured) плагин не показывает блок рекомендуемых товары.

Параметры для настройки блока рекомендуемых товаров:

Параметр Значение по умолчанию Описание
container - jquery селектор для контейнера, в котором нужно разместить рекомендуемые товары (если рекомендуемых товаров нет, то этот контейнер скрывается)
containerClosest - jquery селектор который необходимо скрыть, если рекомендуемых товаров нет (может использоваться для загаловков)
form normal Значения normal(обычная форма), light(упрощенная форма), html-код или jquery селектор
count 2 Значения inline(размещение внутри контейнера) или modal(размещение в модал-окне)
more - jquery селектор для кнопки «показать ещё» (кнопка скрывается, если рекомендуемых товаров нет)
successUrl /finish.jsp?id= url, по которому отправляется пользователь после успешной отправки заявки или false в случае, если редирект не нужен

Примеры: 

<script src="//ajax.googleapis.com/ajax/libs/jquery/1.10.1/jquery.min.js"></script>
<script src="http://img.apishops.org/SinglePageWebsites/custom/js/apishopsForm.2.js"></script>
<link rel="stylesheet" href="http://img.apishops.org/SinglePageWebsites/custom/css/apishopsForm.2.css">
<script>

        $('#containerid').apishopsForm({
            type:'inline', /*тип открытия [inline|modal]*/
            debug:false, /*отладочный режим [true|false]*/
            successUrl:false, /*url благодарственной страницы [url|false]*/
            form:'light', /*тип формы [normal|light|jquery-selector]*/
            theme:3, /*номер темы*/
            siteId:147233, /*айди сайта из панели партнера http://www.apishops.com/Webmaster/WebsiteGroup/WebsiteGroupList.jsp */
            productId:888647,/*айди товара из ассортимента на странице http://www.apishops.com*/
            hidden_fields:['count','fio','address'],
            displayed_containers:['picture','price','name'],
            featured:{
               container: '#featureconatiner', /*контейнер для размещения в нем рекомендуемых товаров*/
               count:3 /*количество загружаемых товаров*/
            }
        });
</script>
<div id=containerid>
</div>
<div id=featureconatiner>
C этим товаром берут
</div>
