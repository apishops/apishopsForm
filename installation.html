Подключение плагина к своим страницам

Вид страницы с подключенными JS и стилями и инициализированным плагином

<html>
    <head>
        <!-- init jquery -->
        <script src="//ajax.googleapis.com/ajax/libs/jquery/1.10.1/jquery.min.js"></script>

        <!-- init apishops -->
        <script src="http://img.apishops.org/SinglePageWebsites/custom/js/apishopsForm.2.js"></script>
        <link href="http://img.apishops.org/SinglePageWebsites/custom/css/apishopsForm2.css" rel="stylesheet">
    </head>
    <body>
        <script>
            $(document).ready( function() {
                $('#container2').apishopsForm({
                        type:'inline', /*тип открытия [inline|modal]*/
                        form:'light', /*тип формы [normal|light|jquery-selector]*/
                        siteId:10221, /*айди сайта из панели партнера http://www.apishops.com/Webmaster/WebsiteGroup/WebsiteGroupList.jsp */
                        productId:632879, /*айди товара из ассортимента на странице http://www.apishops.com*/
                        price:1017.41, /*цена товара*/
                    });
            });
        </script>
        <div id=container2>
        </div>
    </body>
</html>

Поэтапно

Для подключения API необходимо подключить файл стилей (apishopsForm2.css), js файл библиотеки (apishopsForm.2.js), и если на странице не подключен jquery, подключить ещё и jquery (jquery.min.js). После этого можно инициализировать плагин.

1. Подключение стилей

apishopsForm2.css <link rel="stylesheet" href="http://img.apishops.org/SinglePageWebsites/custom/css/apishopsForm2.css">

2. Подключение JavaScript

JQuery <script src="//ajax.googleapis.com/ajax/libs/jquery/1.10.1/jquery.min.js"></script>
apishopsForm.2.js <script src="http://img.apishops.org/SinglePageWebsites/custom/js/apishopsForm.2.js"></script>

3. Инициализация плагина

Инициализация первой формы происходит через вызов jquery плагина $(..).apishopsForm({}). В качестве параметров используется js объект с параметрами.

Код:

$(document).ready( function() {
    $('#container2').apishopsForm({
            type:'inline', /*тип открытия [inline|modal]*/
            form:'light', /*тип формы [normal|light|jquery-selector]*/
            siteId:10221, /*айди сайта из панели партнера http://www.apishops.com/Webmaster/WebsiteGroup/WebsiteGroupList.jsp */
            productId:632879, /*айди товара из ассортимента на странице http://www.apishops.com*/
            price:1017.41, /*цена товара*/
        });
});
