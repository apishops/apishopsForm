Блок отображения информации по бесплатным подаркам

По умолчанию, блок с информацией о бесплатных подарках отображается только тогда, когда для заданного сайта (siteId) и(или) товара (productId) указан один или два подарка, но если подарок не задан, то этот блок не отображается.

Внешний вид блока с информацией по беслптаным подаркам имеет три варианта и указывается параметром gift

За настройку этого блока отвечает всего один параметр gift, который по умолчанию равен 'auto', но может иметь следующие значения:

  • false – никогда не показывать информацию по бесплатным подаркам
  • auto – показывать информацию по бесплатным подаркам только в том случае, если эти подарки заданы
  • [1..3] – показывать информацию по бесплатным подаркам, если эти подарки заданы, в одном из трех вышеуказанных вариантов

Примеры:

Первый вариант блока отображения информации по бесплатным подаркам

<script src="//ajax.googleapis.com/ajax/libs/jquery/1.10.1/jquery.min.js"></script>
<script src="http://img.apishops.org/SinglePageWebsites/custom/js/apishopsForm.2.js"></script>
<link rel="stylesheet" href="http://img.apishops.org/SinglePageWebsites/custom/css/apishopsForm.2.css">
<script>

        $('#containerid').apishopsForm({ callback:1,
            type:'inline', /*тип открытия [inline|modal]*/
            debug:false, /*отладочный режим [true|false]*/
            successUrl:false, /*url благодарственной страницы [url|false]*/
            form:'light', /*тип формы [normal|light|jquery-selector]*/
            theme:3, /*номер темы*/
            siteId:147233, /*айди сайта из панели партнера http://www.apishops.com/Webmaster/WebsiteGroup/WebsiteGroupList.jsp */
            productId:888647,/*айди товара из ассортимента на странице http://www.apishops.com*/
            hidden_fields:['count','fio','address'],
            displayed_containers:['picture','price','name'],
            gift:1, /*внешний вид блока отображения информации по бесплатным подаркам*/
        });
</script>
<div id=containerid>
</div>
<div id=featureconatiner>
C этим товаром берут
</div>
