Параметры

Параметры, это то, что передается в фигурных скобках при инициализации плагина:
    $('...').apishopsForm({
            type:'inline', /*тип открытия [inline|modal]*/
            form:'light', /*тип формы [normal|light|jquery-selector]*/
            siteId:10221, /*айди сайта из панели партнера http://www.apishops.com/Webmaster/WebsiteGroup/WebsiteGroupList.jsp */
            productId:632879, /*айди товара из ассортимента на странице http://www.apishops.com*/
            price:1017.41, /*цена товара*/
        });

Обязательные параметры

Существует всего два параметра, без которых работа плагина невозможна — siteId и productId, для всех остальных параметров плагин для своей работы будет использовать собственные значения по умолчанию
Параметр Описание
siteId айди сайта из панели партнера http://www.apishops.com/Webmaster/WebsiteGroup/WebsiteGroupList.jsp
productId айди товара из ассортимента на странице http://www.apishops.com 
$(document).ready( function() {
    $('#container2').apishopsForm({ callback:1,
            siteId:10221, /*айди сайта из панели партнера http://www.apishops.com/Webmaster/WebsiteGroup/WebsiteGroupList.jsp */
            productId:632879 /*айди товара из ассортимента на странице http://www.apishops.com*/
        });
});

Необязательные параметры

Необязательные параметры, это те, для которых существуют значения по умолчанию
Параметр Значение по умолчанию Описание
type inline Значения inline(размещение внутри контейнера) или modal(размещение в модал-окне)
form normal Значения normal(обычная форма), light(упрощенная форма), html-код или jquery селектор. В случае html-кода или jquery селектора, которые указывает на html, который необходимо использовать, в качестве формы с полями заказа необходимо задать параметр inputs
lang 1 внутренний для Apishops код страны куда будет оформлен заказа (1 — Россия, 6 — Украина, 7 – Беларусь, 8 – Казахстан)
theme 0 порядковый номер темы (более подробно в разделе темы)
gift 'auto' порядковый номер темы (более подробно в разделе подарки), false — чтобы отключить и 'auto', автоматически подбор темы
callback 0 иконка вызова обратного звонка (0 — не показывать, 1 — показывать)
featured {} набор параметров для отображения рекомендуемых товаров (более подробно в разделе рекомендуемые товары)
hidden_fields [] Массив из названий полей, которые не надо показывать. Доступные названия: 'address','count','delivery','email','promocode','fio','payment','phone','region','cost'. Например, чтобы спрятать поля адрес доставки и промокода, значение hidden_fields будет ['address','promocode']
displayed_containers [] Массив из названий частей формы, которые надо показывать. Доступные значения 'picture', 'price', 'name', 'quickview'. По умолчананию контейнеры с картинкой, ценой, названием товара и ссылкой на подробное описание скрыты.
page {} Объект c price и oldprice переменных, которые должны указывать на CSS селекторы контейнеров актуальной и старой цены сооветвственно.

Пример использования:
page:{price:'.apishopsProductPrice',oldprice:'.apishopsProductOldPrice'}
successUrl /finish.jsp?id= url, по которому отправляется пользователь после успешной отправки заявки или false в случае, если редирект не нужен
