Регулировка блока рекомендованых товаров в готовых лендингах

Блоки рекомендованных товаров в готовых лендингах работают по тому же принципу, который описан в разделе подключения блока к собственным страницам. Т.е. во всех шаблонах обязательно присуствуют:

  • Инициализации плагина для подключения форм с параметрами , которые можно редактировать(скриншот)
  • Включенный параметр featured.container, указывающий на контейнер для размещения рек.товаров (как правило .__apishopsFormFeaturedFormContainer__(скриншот) или #featuredFormInsertContainer (скриншот))
  • Контейнер, в котором размещаются рек.товары (скриншот)

С блоком можно производить следующие манипуляции:

+ Изменить положение блока

Чтобы поменять местоположение блока, необходимо знать, на какой контейнер указывает параметр featured.container
, затем найти этот блок в HTML шаблоне

и перенести его в то место шаблона, которое вы находите нужным

+ Изменить количество отображаемых товаров

Чтобы изменить количество отображаемых товаров, необходимо найти в шаблоне параметр featured
, и добавить параметр featured.count, равный желаемому количеству отображаемых товаров

+ Изменить тему блока

Чтобы поменять тему для отображаемых товаров, необходимо найти в шаблоне параметр featured

, и установить параметр featured.theme, равный порядковому номеру одной из наших тем, а параметр featured.form равный light

+ Настроить стили внешнего вида блока

Чтобы настроить внешний вид блока необходимо знать, что во всех лендингах ApiShops блоки рекомендуемых товаров формируются двумя способами: на основе доступных в конструкторе тем(если в шаблоне можно найти #featuredFormInsertContainer) или на основе HTML кода блока в шаблоне (если в шаблоне можной найти __apishopsFormFeaturedFormContainer__).

  • Для изменения внешнего вида блоков блоков, которые формируются на основе тем

    необходимо найти параметр featured.theme в шаблоне

    выбрать эту тему в конструкторе и с помощью иконок настроить внешний вид — в нижней части экрана конструктор сгенерирует необходимые стили

    которые вам останется всего лишь вставить в CSS шаблон лендингах

  • Стили для блоков, которые формируются на основе HTML кода эти блоков в шаблоне

    заданы в CSS шаблоне лендинга и там же легко находятся и редактируются, как и внешний вид других частей шаблона
